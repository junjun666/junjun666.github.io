2:找到本地下例如C:\xampp\htdocs文件夹下，在下面操作php

使用php访问数据库

1)连接数据库
mysqli_connect(host,user,pwd,dbname,port);
一般host为本机服务器地址:127.0.0.1
2)先提交sql数据的UTF8
$sql = “SET NAMES UTF8”;
mysqli_query($conn,$sql);
3）再提交sql数据
$sql = INSERT INTO…
$result = mysqli_query($conn,$sql);
4)查看返回结果
if($result){
echo …;
}else{
echo …;
}
